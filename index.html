compatibility

This template is a pure CSS layout. It validates successfully with www.w3.org as XHTML 1.0 Strict. Since it's tableless, and the element/font sizes are defined relatively, it gracefully increases/decreases in size for different resolutions and text sizes. It renders consistently in:

  • Firefox 1.0
  • Netscape 7.02
  • Opera 7.54
  • Internet Explorer 6.0
